# Gym Landing Page

A simple gym landing page built with React.  
This project was created to practice React fundamentals and understand how components, state, and props work together.

## Project Overview
The landing page represents a basic gym website layout.  
It focuses on structuring the UI using reusable React components rather than connecting to any backend or storing data.

The main goal of this project was learning React basics in a practical way.

## Features
- Component-based structure
- Reusable UI components
- Popup subscription form
- State handling using `useState`
- Props usage for data flow between components
- Clean and simple layout

## Technologies Used
- React
- JavaScript
- CSS

## Technical Highlights
- Built entirely using React components
- State management for UI interactions (popup form)
- Clear separation of components and responsibilities
- Focus on readability and maintainable structure

## Challenges
The main challenge was organizing the UI into reusable components and managing component communication using props while keeping the structure simple and understandable.

## Project Purpose
This project was built for learning purposes, focusing on React fundamentals such as components, props, and state.
